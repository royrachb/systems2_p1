C++ Graph Algorithms Project
Build Instructions

    Ensure you have a C++ compiler (e.g., g++).
    Navigate to the project directory.
    Run make to compile the project.
    Run make test to compile the project tests.

Run Instructions

    To run the demo program:

    bash

./demo

To run the tests:

bash

    ./test

Files

    Algorithms.cpp: Implementation of graph algorithms.
    Algorithms.hpp: Header file for the algorithms.
    Demo.cpp: Demonstration program.
    Graph.cpp: Implementation of the Graph class.
    Graph.hpp: Header file for the Graph class.
    Test.cpp: Unit tests.
    TestCounter.cpp: Additional tests.
    doctest.h: Testing framework.
    Makefile: Build configuration.
